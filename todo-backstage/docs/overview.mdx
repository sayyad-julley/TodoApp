## Quick Start

<Steps>
  <Step title="Clone Repository">
    ```bash
    git clone https://github.com/sayyad-julley/TodoApp
    cd TodoApp/todo-backstage
    ```
  </Step>
  <Step title="Install Dependencies">
    ```bash
    yarn install
    ```
  </Step>
  <Step title="Configure Environment">
    ```bash
    cp app-config.example.yaml app-config.local.yaml
    # Edit app-config.local.yaml with your settings
    ```
  </Step>
  <Step title="Start Development">
    ```bash
    yarn start
    ```
  </Step>
</Steps>

## Development Workflow

1. **Local Development**: Use `yarn start` for local development
2. **Template Development**: Create and test templates in `examples/template/`
3. **Plugin Development**: Extend functionality with custom plugins
4. **Documentation**: Update docs in the `docs/` directory
5. **Testing**: Run tests with `yarn test`

## Configuration

### Environment Variables

```bash
# GitHub Integration (required)
GITHUB_TOKEN=your-github-personal-access-token

# Traycer AI (optional)
TRAYCER_API_URL=https://api.traycer.ai
TRAYCER_API_KEY=your-traycer-api-key

# Database (default: SQLite)
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
POSTGRES_USER=postgres
POSTGRES_PASSWORD=password
```

### App Configuration

Main configuration in `app-config.yaml`:

```yaml
# Core configuration
app:
  title: Backstage Template

# Database configuration
database:
  client: better-sqlite3
  connection: ':memory:'

# GitHub integration
integrations:
  github:
    - host: github.com
      token: ${GITHUB_TOKEN}

# Catalog locations
catalog:
  locations:
    - type: file
      target: ../../examples/template/golden-path-todo/template.yaml
      rules:
        - allow: [Template]
```
